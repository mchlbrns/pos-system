# 🇵🇭 Universal POS System v2

Welcome to your new POS System! This application has been designed specifically for shop owners (Water Stations, Laundry Shops, Motorcycle Repair, etc.) to be setup and run in under 5 minutes without needing any technical knowledge.

---

## 🚀 Easy 3-Step Setup

### Step 1: Install Node.js
If you haven't installed Node.js yet:
1. Go to **[https://nodejs.org/](https://nodejs.org/)**
2. Download the Recommended version (**LTS**) for Windows.
3. Install it using the default settings.

---

### Step 2: One-Click Installation
1. Double-click the **`install.bat`** file in this folder.
2. A window will open and automatically prepare the database and install everything.
3. Once finished, press any key to close the window.

---

### Step 3: Start the POS
1. Double-click the **`start.bat`** file in this folder.
2. The system will start, and it will automatically open your web browser to **`http://localhost:5173`**.
3. **Important:** Keep the system window open while using the POS. You can minimize it.

---

## 📖 Admin Controls (PIN Protection)
To change products, switch your business type, or configure the printer:
- Click the **Admin** tab.
- Enter the default administrative PIN: **`1234`**

---

## 🖨️ Printer Setup
1. Go to the **Admin** dashboard.
2. Click **Printer Setup**.
3. The setup wizard will automatically find your USB and network thermal or impact printers.
4. Select your printer and click **Print Test Receipt** to verify.
